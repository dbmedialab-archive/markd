# Markd

Super duper light-weight customizable jquery-plugin markdown-editor.

> Awsomesauce!

* Works with textareas and (alsmost) any «contenteditable»
* No editing inside iframe-crap
* Customize to your hearts delight

###### Keyboard-commands
* `ctrl+p` display a preview the current text
* `ctrl+f` enter fullscreen-mode with live preview
* `ctrl+h` opens a new window with syntax reference http://daringfireball.net/projects/markdown/syntax
* `ctrl+b` makes current selection bold
* `ctrl+i` makes current selection italic
* `ctrl+k` makes current selection a link
